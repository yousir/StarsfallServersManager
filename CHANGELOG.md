# Changelog

## [1.0.0] - 2025-09-01
### Added
- 新增 `rz` 和 `sz` 文件上传和下载功能

### Fixed
- 修复 SSH 终端提示符常用的快捷键问题

## Fixed 2025-09-12
- 修复 SSH 终端提示符常用的快捷键问题，
    1. tab 键补全功能修复
    2. Ctrl+C 终止执行
    3. shift+ins 粘贴功能修复与增强
    4. 历史命令记录功能修复
- 增强 xterm 终端多行命令功能
    1. 支持用户在多行命令的情况下进行跨行移动，
    2. 支持中间插入粘贴多行命令参数，
    3. 支持删除任意行的命令参数，任意位置粘贴命令参数
